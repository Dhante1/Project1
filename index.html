What is "Caesar Cipher?"

The Caesar Cipher is a simple and well-known encryption technique named after Julius Caesar, who used it for secure communication. It works by shifting each letter in the original message a fixed number of places down or up the alphabet.

Shift Value: if Caesar Cipher's rule Each letter in the message is shifted by a random value between 1 and 25. For example, with a shift of 3, 'A' becomes 'D', 'B' becomes 'E', and so on. But to make things more unique This one is you are going to shift BACKWARDS.

Letter Wrapping: If the shift causes a letter to go past 'Z', it wraps around to the beginning of the alphabet (e.g., 'Z' shifted by 1 becomes 'A'). Uppercase Conversion: The input message is converted to uppercase before encryption for consistency.
